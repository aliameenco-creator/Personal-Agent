# Brandify AI

AI-powered creative studio for rebranding, LinkedIn posts, YouTube content, and thumbnail generation.

Built with React + TypeScript + Google Gemini

</div>

## Features

### Rebrand Module
- **Batch image rebranding** - Upload up to 10 images and rebrand them all at once
- **Smart logo placement** - Provide white and black logo variants; AI picks the best one based on background contrast
- **Brand name replacement** - Automatically finds and replaces existing brand text
- **Iterative editing** - Chat-based refinement to fine-tune results after initial rebrand
- **Download options** - Single image download or batch ZIP export

### YouTube Studio Module
A step-by-step workflow for complete YouTube video optimization:

1. **Title Generation** - Generate 5 title candidates from your video topic (supports voice input)
2. **Description** - AI-generated descriptions with guaranteed key phrase insertion (for SEO via vidIQ)
3. **Tags** - Hybrid AI + algorithmic tag generation with YouTube's 500-character limit enforcement
4. **Thumbnails** - AI-generated 16:9 thumbnails using Gemini's image model
5. **Video Hooks** - Upload an SRT file to generate motion graphics clips using Google Veo 3.1

### LinkedIn Studio Module
Generate single image posts or multi-slide carousel posts for LinkedIn:

- **Single image or carousel** - Choose between a single post image (1080x1350) or a multi-slide carousel (2-10 slides)
- **AI content generation** - Provide a topic and AI generates slide-by-slide content (headlines, body text, layout suggestions)
- **Editable slides** - Review, edit, and reorder AI-generated content before visual generation
- **Reference post styling** - Upload screenshots of posts you like and AI matches the visual style
- **Parallel image generation** - All slides generate simultaneously for faster results
- **Per-slide editing** - Click the edit icon on any slide to refine it with a text prompt
- **PDF carousel export** - Download all slides as a single PDF file ready for LinkedIn carousel upload
- **Persistent brand kit** - Save your brand identity (logos, colors, tagline, profile photo, social handle, font preference) once and reuse across sessions
- **Brand kit toggle** - Option to generate posts without any branding applied

### YouTube Thumbnail Generator
A dedicated tool for creating custom YouTube thumbnails:

- **Subject image upload** - Upload your face or main subject for the thumbnail
- **Style reference images** - Upload up to 3 reference thumbnails to match their visual style
- **Voice-to-text prompting** - Use your microphone to dictate the thumbnail description (transcribed via Gemini)
- **16:9 output** - Generates standard 1280x720 YouTube thumbnail dimensions
- **Click to zoom** - Full-screen preview of generated thumbnails
- **One-click download** - Download the generated thumbnail as PNG

## AI Content Provenance & Metadata Stripping

Google Gemini embeds **C2PA metadata** (Coalition for Content Provenance and Authenticity) into every image it generates. C2PA is an industry-standard digital watermark that declares content as AI-generated. Social media platforms like LinkedIn read this metadata and display a "Created with AI" label on uploaded images.

### How we handle it

All images generated by Gemini pass through a **Canvas re-encoding step** before being returned to the app. This works by:

1. Drawing the base64 image onto an HTML `<canvas>` element
2. Exporting it back via `canvas.toDataURL()`
3. The exported image contains only raw pixel data - all metadata (C2PA, EXIF, IPTC, XMP) is stripped

This happens at the API response level in all image-producing services, so every downstream action (display, download, ZIP export, PDF export) uses clean images.

**Files involved:**
- `utils/stripMetadata.ts` - The re-encoding utility
- `services/geminiService.ts` - Rebrand & edit image functions
- `services/youtube/thumbnailService.ts` - YouTube Studio thumbnail generation
- `services/linkedin/slideGeneratorService.ts` - LinkedIn slide generation
- `services/thumbnail/thumbnailGenService.ts` - Standalone thumbnail generation
- `services/ai/geminiTextProvider.ts` - Generic image generation

- **Frontend** - React 19 + TypeScript 5.8

- **AI** - Google Gemini SDK (`@google/genai`)
- **Styling** - Tailwind CSS (CDN)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in `.env.local`:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. Run the app:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:3000`.
